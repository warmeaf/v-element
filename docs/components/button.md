# Button 按钮

常见的操作按钮。

## 基础用法

使用 `type`、`size`、`plain`、`round` 和 `circle` 属性来定义按钮的样式。

<script setup>
  import VButton from '../../src/components/button/Button.vue'
</script>

<v-button>button</v-button>
<v-button loading>loading</v-button>
